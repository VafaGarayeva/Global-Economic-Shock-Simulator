# Global-Economic-Shock-Simulator
An interactive data analytics project that simulates the impact of global economic shocks on macro-indicators and individual financial decisions. Developed using Python for data engineering and Tableau for high-fidelity historical storytelling
🌍 Global Economic Shock Simulator
An Interactive Data Experience: How Global Crises Shape Our World and Your Decisions
📌 Project Overview
The Global Economic Shock Simulator is an end-to-end data analytics project that visualizes the ripple effects of major historical economic crises (such as the 1973 Oil Shock, 2008 Financial Crisis, and COVID-19).

This project goes beyond static charts by allowing users to step into the shoes of different stakeholders—Bankers, Investors, or Policy Makers—to see how individual decisions resonate with global economic indicators like Inflation, Unemployment (ILO estimates), and GDP Growth.

🛠 Tech Stack
Data Processing: Python (Pandas, NumPy)

Analysis: SQL & Predictive Modeling

Visualization: Tableau (Interactive Dashboards)

Data Sources: World Bank (WDI), International Labour Organization (ILO), IMF.

🚀 Key Features
Interactive Time Machine: A global map animation showing the spread of economic shocks from 1970 to 2024.

Role-Based Simulation: Users select a persona (e.g., Small Business Owner vs. Central Banker) and make high-stakes decisions during a crisis.

ILO-Standardized Analytics: Utilizing ILO modeled estimates for unemployment to ensure cross-country comparability and accuracy.

Sectoral Impact Analysis: Visualizing how different industries (Tech, Energy, Trade) recover at different speeds post-crisis.

📊 Methodology (The Data Journey)
Extraction: Scraped and downloaded macroeconomic datasets from global financial institutions.

Cleaning & Transformation: Performed complex "Pivoting" and "Melting" in Python to convert Wide-format World Bank data into Long-format for Tableau optimization.

Modeling: Built a logic-based engine to simulate "Crisis Probability" based on historical inflation and debt thresholds.

Dashboarding: Crafted a high-fidelity Tableau dashboard with a focus on "Minimalist & Serious" aesthetics.

💡 Key Insights
Recovery Lag: Demonstrated that while GDP often recovers within 2 years, Unemployment rates (ILO) take an average of 5 years to return to pre-crisis levels.

Decision Resonance: Users learn that "Reducing Expenses" at a micro-level can sometimes accelerate a "Global Liquidity Trap" at the macro-level.
